# SQL com PostgreSQL – Prática Profissional

Este repositório contém exercícios práticos de SQL realizados em ambiente PostgreSQL,
com foco em uso real de mercado.

O objetivo é demonstrar conhecimento aplicado em consultas, relatórios,
relacionamentos entre tabelas e boas práticas de banco de dados.

## Tecnologias utilizadas
- PostgreSQL
- pgAdmin
- SQL

## Conteúdo do laboratório

### Fase 1 – Criação de usuários
Criação de usuários simulando aplicação e auditoria.

### Fase 2 – Dados sensíveis
Criação de tabela contendo dados pessoais (nome, CPF e email).

### Fase 3 – Privilégios excessivos
Simulação de erro comum com concessão ampla de privilégios em tabela sensível.

### Fase 4 – Menor privilégio
Correção aplicando o princípio do menor privilégio e auditoria de permissões.

### Fase 5 – SELECT básico
Consultas SQL utilizando WHERE, ORDER BY, LIMIT e COUNT.

### Fase 6 – GROUP BY
Geração de relatórios com agregação de dados por cidade.

### Fase 7 – JOIN
Criação de tabelas relacionadas e consultas utilizando JOIN e LEFT JOIN
para consolidação de dados entre clientes e pedidos.

## Objetivo profissional
Este laboratório foi desenvolvido para fins de estudo e demonstração prática
de SQL aplicado a contextos reais de trabalho.
