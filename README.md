# SQL com PostgreSQL – Prática Profissional

Este repositório contém exercícios práticos de SQL realizados em ambiente PostgreSQL, com foco em cenários reais de mercado, incluindo segurança, controle de acesso e análise de dados.

O objetivo é demonstrar conhecimento aplicado em bancos de dados relacionais, consultas SQL, relatórios e boas práticas de segurança.

---

## Tecnologias utilizadas
- PostgreSQL
- pgAdmin
- SQL

---

## Estrutura do projeto

### Fase 1 – Criação de usuários
Criação de usuários simulando uma aplicação e um perfil de auditoria.

### Fase 2 – Dados sensíveis
Criação de tabela contendo dados pessoais (nome, CPF e email), simulando um cenário real de dados sensíveis.

### Fase 3 – Privilégios excessivos
Concessão de permissões amplas para demonstrar riscos de segurança.

### Fase 4 – Menor privilégio
Correção das permissões aplicando o princípio do menor privilégio.

### Fase 5 – Consultas básicas
Consultas SQL utilizando SELECT, WHERE, ORDER BY, LIMIT e COUNT.

### Fase 6 – Relatórios com GROUP BY
Consultas de agregação para geração de relatórios.

### Fase 7 – Joins e relacionamentos
Relacionamento entre tabelas e consultas com JOIN.

---

## O que este projeto demonstra

- Criação e gerenciamento de usuários no PostgreSQL
- Controle de permissões e segurança em banco de dados
- Aplicação do princípio do menor privilégio
- Escrita de consultas SQL para análise de dados
- Criação de relatórios com GROUP BY
- Relacionamento entre tabelas utilizando JOIN
